#  Bacadem - Site Vitrine Multilingue

Plateforme professionnelle multilingue (FR, RU, EN) avec système sécurisé de visualisation de documents PDF.
Utilisation de PDF.js
Visitez le site sur (https://bacadem.org)

##  Points forts
- **Gestion Multilingue** : Système de traduction dynamique.
- **Protection PDF** : Visualisation sécurisée empêchant l'accès direct aux fichiers sources.
- **Design Responsive** : Interface fluide pour la lecture de documents sur tablette et mobile.
- **Framework CSS** : Bootstrap 5 (Design responsive et composants UI)

##  Stack Technique
- **Backend** : PHP 8
- **Frontend** : JavaScript, CSS3
- **Sécurité** : Contrôle des headers HTTP pour la protection des fichiers.
Utilisez le code avec précaution.
